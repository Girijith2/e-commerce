This project is a backend REST API developed using **Spring Boot** for managing an e-commerce style cart system. The application handles core functionalities such as managing products, product images, product reviews, and customer orders. It follows a layered architecture using **Controller, Service, Repository, and Entity** layers to maintain clean separation of concerns. **Spring Data JPA** is used for database operations and DTOs are implemented to transfer data safely between layers. The system also supports dynamic product filtering and structured order creation with multiple order items.

In the future, the project can be improved by adding **user authentication and authorization using JWT**, implementing a **shopping cart module**, adding **pagination and sorting for product listings**, integrating **Swagger for API documentation**, and improving validation and security features. These improvements will make the system more scalable, secure, and production-ready.
